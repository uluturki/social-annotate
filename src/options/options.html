Why we created this tool?

Human annotated data is a crucial component in the analysis of online systems. Most research efforts go through cumbersome data collection phases where they either build complicated annotation pipelines for online data or annotate offline data shared on spreadsheets or surveys. These approaches introduce significant overhead to any study requiring data collection and causes researchers to divert significant effort away from analysis and modelling to data collection, especially when they have limited programming experience.

We aim to develop a tool that can easily adapt to different tasks and extend to most online platforms. This tool allows collection of data directly on the online platform as a user sees and interacts with it as they normally would by injecting surveys into webpages on the browser, and annotations can easily be exported as files or sent to API endpoints. We hope to promote and expedite annotation of online datasets and reduce the data collection overhead so that researchers can focus on research by developing this tool.

How to modify options?

To modify survey, please make changes in the Options page. Each different survey is listed under "Survey information" section. Form objects created for each annotation follows a format indicated by jsonform package. Once you update your form object, please make sure to validate the format using the link on Options page.

How to add new social media platform?

For detailed instructions please refer to our paper. We also encourage sharing your "recipes" with the project by creating pull requests. We would like to organize those recipes under "recipes" folder.
